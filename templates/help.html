Calendar Type

This defines if the calendar you want is for a team or a competition.

OneFootball ID

This should be the ID for the team/competition in OneFootball.
First, go to onefootball.com and find the team or competition you want on the search bar.
Then, when you're on the main page for the team/competition, you can find the ID on the URL.

For example, https://onefootball.com/en/team/atletico-mineiro-1683 goes to the Atlético-MG page.
From this URL, we learn the ID for Atlético is "atletico-mineiro-1683".

Event Length

You can use this option to decide how long each match should be in your calendar.
By default, we use 120 minutes for each calendar event. If you want to use this length, you don't need to include it in your URLs.

Calendar Integration

After you have your OneFootball ID and know your event length, you can use the following URL templates to add it to your calendar:
  • For a team calendar: {{ url_for('create_team_calendar', team_id='ONEFOOTBALL_ID', event_length='EVENT_LENGTH', _external=True)}}
  • For a competition calendar: {{ url_for('create_competition_calendar', comp_id='ONEFOOTBALL_ID', event_length='EVENT_LENGTH', _external=True)}}
For example, to get the calendar for Atlético's schedule with the default event length you can use:
{{ url_for('create_team_calendar', team_id='atletico-mineiro-1683', _external=True)}}

Alternatively, you can download the .ics file and manually import it to your calendar.
