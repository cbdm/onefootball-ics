Team ID

This should be the ID for the team in OneFootball.
For example, https://onefootball.com/en/team/atletico-mineiro-1683 goes to the Atlético-MG page.
From this URL, we learn the ID for Atlético is "atletico-mineiro-1683".

Event Length

You can use this option to decide how long each match should be in your calendar.
By default, we use 120 minutes for each calendar event. If you want to use this length, you don't need to include it in your URLs.

Calendar Integration

After you have your team ID and know your event length, you can use the following template to use it in your calendar:
{{ url_for('create_team_calendar', team_id='TEAM_ID', event_length='EVENT_LENGTH', _external=True)}}

For example, to get the calendar for Atlético's schedule with the default event length you can use:
{{ url_for('create_team_calendar', team_id='atletico-mineiro-1683', _external=True)}}

Alternatively, you can download the .ics file and manually import it to your calendar.
However, that will not automatically be updated, like the URL should be. {% endblock %}
